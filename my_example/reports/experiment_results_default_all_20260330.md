## 3. 结果解读

### 3.1 排名

| 维度 | 排序 |
| --- | --- |
| M2 收益 | `Genetic > SA > Improved Greedy > EWF > Random` |
| 运行时间 | `Random ≈ EWF < Improved Greedy << SA < Genetic` |
| M3 仰角 | `Improved Greedy > Genetic ≈ EWF > Random > SA` |

### 3.2 关键信息

| 算法 | 主要定位 | 结论 |
| --- | --- | --- |
| Random | 最低下界 | 明显落后，证明调度顺序设计有价值 |
| EWF | 基础贪心 | 能稳定提升 M1，但收益选择能力较弱 |
| Improved Greedy | 主基线 | 速度快，M2 和 M3 都较均衡 |
| SA | 中间层方法 | 比改进贪心更高收益，但耗时上升明显 |
| Genetic | 全局搜索 | M2 最优，但计算代价最高 |

### 3.3 最适合写进论文的结论

1. `Random` 明显落后，说明调度顺序和任务筛选是必要的。
2. `EWF` 能快速拿到可行任务，但收益优化能力有限。
3. `Improved Greedy` 是很强的工程型主基线，兼顾速度、收益和观测质量。
4. `SA` 提供了介于启发式和 GA 之间的折中方案。
5. `Genetic` 的收益最高，但需要承担最大的计算开销。
